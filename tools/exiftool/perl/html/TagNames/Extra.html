Extra Tags

The extra tags represent extra information extracted or generated by ExifTool that is not directly associated with another tag group. The Group column lists the family 1 group name when reading. Tags with a "-" in this column are write-only.

Tags in the family 1 "System" group are referred to as "pseudo" tags because they don't represent real metadata in the file. Instead, this information is stored in the directory structure of the filesystem. The four writable "pseudo" tags (FileName, Directory, FileModifyDate and FileCreateDate) may be written without modifying the file itself.
